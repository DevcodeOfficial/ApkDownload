# ApkDownloader 

You Can Easily Download Apk Files Using This Api

## installing

```sh
$ pip install apkdownload
```

## How To Use

```python
import apkdownload as apk

apk.dl('App Name', 'Directory')
```

## example

```python
import apkdownload as apk

apk.dl('Cpu-z', '/sdcard/download)
```

## example

```python
import apkdownload as apk

apk.dl('Subway Surf', '/Desktop)
```
